## 2. 구현 과정
1) 다익스트라 최단거리 알고리즘 관련 클래스 생성
   - DistanceGraph : 그래프의 간선이 거리인 클래스
   - TimeGraph : 그래프의 간선이 시간인 클래스
2) 지하철 노선도 dummy data 세팅
   - 존재하는 지하철역 세팅
   - 존재하는 호선 세팅
   - 호선과 각 호선마다 존재하는 지하철역의 관계 세팅
   
## 3. 느낀점
- 갑자기 난이도가 확올라감
- 다익스트라 같은 알고리즘도 누군가 라이브러리로 다 만들어 놓았다는 것을 처음 알게됨
